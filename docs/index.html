PokeScanner · Cardmarket Barcode Pricer

Static GitHub Pages scanner. Scan with a keyboard-style 1D scanner or use the camera scanner. Resolves local sealed-product candidates, shows cached/public prices, and opens Cardmarket pages.

Scan / enter barcode

Camera scanning uses the browser BarcodeDetector API where available. Works best in Chrome/Edge on HTTPS. If unsupported, use a USB scanner or manual entry.

Your scan → select memory

Selections and unknown-barcode mappings are stored locally in this browser. If a barcode is ambiguous, choose Remember & Open; next scan opens that exact Cardmarket page automatically.

Optional: online barcode resolver API

If a barcode is not in the local mapping, PokeScanner can call your Google Apps Script resolver. It returns Cardmarket candidates automatically, so you do not have to manually find product pages.

Optional: external daily export

If your Google Sheet/App Script publishes JSON with newer price snapshots, paste its URL here. It must allow browser CORS and use the same shape as cardmarket_price_fixtures.json or a simple {url: {text: "..."}} map.

Important: GitHub Pages cannot use your logged-in Cardmarket session or scrape pages server-side. It can fetch Cardmarket's public daily price guide if CORS allows it, use bundled cached snapshots, and open the real Cardmarket page in your browser.
For unknown barcodes, choose/save the product once in the browser. Export local mappings later if you want to commit them into data/barcode_aliases.json.
