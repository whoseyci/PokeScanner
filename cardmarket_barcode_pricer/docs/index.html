Cardmarket Barcode Pricer

Static GitHub Pages scanner. Scan an EAN/UPC, resolve local sealed-product candidates, show cached/exported prices, and optionally open the strongest Cardmarket product page.

Scan / enter barcode

Most USB 1D barcode scanners behave like a keyboard: digits + Enter. Auto-open is the most reliable Cardmarket workflow because GitHub Pages cannot bypass Cardmarket CORS/login/bot protections.

Optional: load external daily export

If you publish a Google Sheet/App Script JSON with newer price snapshots, paste its URL here. It must allow browser CORS and use the same shape as cardmarket_price_fixtures.json or a simple {url: {text: "..."}} map.

Important: This static page does not live-scrape Cardmarket. It uses cached/exported price text and opens Cardmarket in your browser for the real page. For live German→German filtered prices, plug in an official API, logged-in local fetcher, or your daily Google Sheet export.
Built for sealed-product barcode triage. Do not silently trust ambiguous barcodes: choose the packaging candidate that matches the item in hand.
